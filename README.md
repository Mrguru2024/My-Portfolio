## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Mrguru2024/My-Portfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd My-Portfolio
    ```
3. Open `index.html` in your browser to view the site.
